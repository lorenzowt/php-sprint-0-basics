# Bookstore HTML

A simple bookstore HTML project for the PHP Full-Stack Sprint 0 Basics course.

## Overview

This project demonstrates basic HTML structure and layout for a bookstore website.

## Features

- Product listing with book details
- Simple navigation
- Responsive layout
- Clean and organized code structure

## Getting Started

1. Open the HTML file in your web browser
2. Browse through the bookstore catalog
3. View book information and details

## Project Structure

- `index.html` - Main bookstore page

## Technologies

- HTML5
- CSS3
